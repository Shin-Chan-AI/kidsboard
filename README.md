# 📚 KidsBoard - 우리 아이 교육 대시보드

안녕하세요! KidsBoard는 아이들의 복잡한 교육 일정과 활동을 한눈에 관리할 수 있도록 도와주는 간단한 웹 대시보드입니다.

---

## ✨ 주요 기능

*   **📅 주간 스케줄:** 요일별, 시간별 학원, 과외 등 고정 일정을 시각적으로 관리합니다.
*   **🗓️ 주요 일정:** 소풍, 시험, 발표회 등 일회성 주요 이벤트를 등록하고 D-Day를 확인합니다.
*   **🎒 준비물:** 학교 준비물을 날짜별로 기록하고 완료 여부를 체크합니다.
*   **📝 숙제:** 과목별 숙제를 마감일, 우선순위에 따라 관리하고 상태를 추적합니다.
*   **📰 교육 뉴스:** 아이의 연령대에 맞는 교육 뉴스를 자동으로 수집하여 보여줍니다.

---

## 💻 기술 스택

*   **Backend:** Node.js, Express.js
*   **Database:** SQLite
*   **Frontend:** HTML, Tailwind CSS, Vanilla JavaScript
*   **Deployment:** Google Cloud Run
*   **DB Backup:** Google Cloud Storage

---

## 🚀 로컬에서 실행하기

1.  **저장소 클론:**
    ```bash
    git clone https://github.com/Shin-Chan-AI/kidsboard.git
    ```

2.  **프로젝트 폴더로 이동:**
    ```bash
    cd kidsboard
    ```

3.  **의존성 설치:**
    ```bash
    npm install
    ```

4.  **서버 실행:**
    ```bash
    npm start
    ```
    서버가 실행되면 브라우저에서 `http://localhost:3000` 으로 접속할 수 있습니다.
